# 아이쑥크림(icecream) 웹 디자인 프롬프트

## 0. 프롬프트 사용법
아래 내용을 디자인 생성 도구(v0, Figma AI, Claude 등)나 디자이너 브리프에 그대로 사용하세요. 한 화면씩 생성할 때는 "전체 컨셉 + 해당 화면 섹션"만 잘라 붙이면 됩니다.

## 1. 서비스 한 줄 정의
유아기(만 4~6세) 자녀의 체력을 국민체력100 공식 기준으로 진단하고, 근처 인증센터의 무료 측정으로 연계하며, 성장 추이를 추적하는 모바일 우선 반응형 웹 서비스.

## 2. 브랜드 & 톤앤매너
- 브랜드명: **아이쑥크림(icecream)** — 항상 소문자 표기. 로고/헤더에서도 소문자 유지
- 타깃 사용자: 만 4~6세 자녀를 둔 30대 학부모 (사용자는 부모, 주인공은 아이)
- 무드: 부드럽고 다정하며 신뢰감 있는. 아이스크림처럼 말랑하고 따뜻하되, 국가 공인 기준을 다루는 만큼 유치하지 않고 정돈된 느낌
- 등급 시스템을 브랜드 자산으로 활용: 씨앗 → 새싹 → 꽃 → 열매 (성장 서사). 각 등급을 귀여운 일러스트/아이콘으로 시각화
- 컬러: 파스텔 톤 중심(민트·크림·연한 코랄). 등급별 컬러 코딩(씨앗=연갈색, 새싹=연두, 꽃=코랄/핑크, 열매=따뜻한 레드/오렌지). 배경은 화이트~크림으로 여백을 넉넉히
- 타이포: 둥근 산세리프(가독성 높고 친근한). 한글 본문은 Pretendard 계열 권장
- 형태: 라운드가 큰 카드, 부드러운 그림자, 넉넉한 패딩. 날카로운 각 지양

## 3. 레이아웃 원칙
- 모바일 우선(360~430px 기준 설계 후 반응형 확장). 하단 탭 내비게이션 사용
- 하단 탭 4개: 홈 / 진단 / 기록 / 센터
- 한 화면에 한 가지 핵심 행동. 학부모가 짧은 시간에 입력·조회하는 맥락이므로 스크롤과 입력 부담 최소화
- 손가락 터치 영역 넉넉히(최소 44px). 숫자 입력은 큰 스텝퍼/키패드 친화적으로

## 4. 화면별 명세

### 4.1 온보딩 / 자녀 프로필 등록 (ACC-01, ACC-02)
- 첫 진입: 서비스 한 줄 소개 + "우리 아이 등록하고 시작하기" CTA
- 자녀 등록 폼: 애칭, 성별(남/여 토글), 생년월(개월수 자동 계산·표시)
- 개월수가 48~83개월 범위를 벗어나면 인라인 안내 문구("유아기 측정 대상 범위 밖이에요")
- 여러 자녀 등록 가능 → 상단에 자녀 전환 셀렉터(아바타 칩)

### 4.2 홈 (대시보드)
- 상단: 선택된 자녀 아바타 + 현재 종합 등급 배지(씨앗~열매 일러스트 크게)
- 최근 측정 요약 카드: 마지막 측정일, 강점 1~2개 / 약점 1~2개 태그
- 주요 액션 카드: "체력 진단하기", "근처 센터 찾기", "우리 동네 참여 현황"(인사이트 카드)
- 약점이 있으면 "이런 활동 어때요?" 추천 콘텐츠 미리보기

### 4.3 체력 진단 — 측정값 입력 (DIA-01)
- 측정 유형 선택 토글: **정식 측정 / 자가측정(참고용)**. 자가측정 선택 시 '참고용' 배지 상시 노출
- 8개 항목 입력 카드(항목명 + 단위 + 큰 입력 필드):
  심폐지구력(10m 왕복오래달리기, 회) / 근력(상대악력, %) / 근지구력(윗몸말아올리기, 회) / 유연성(앉아윗몸앞으로굽히기, cm) / 민첩성(5m×4 왕복달리기, 초) / 순발력(제자리멀리뛰기, cm) / 협응력(3×3 버튼누르기, 초) / 신체조성(BMI)
- 일부 항목만 입력해도 진행 가능(미입력 허용). 각 항목에 "측정 방법 보기" 보조 링크
- 하단 고정 "진단 결과 보기" 버튼

### 4.4 체력 진단 — 결과 (DIA-02, DIA-03)
- 상단: 종합 등급 배지(씨앗~열매) + 축하/격려 톤의 한 줄 코멘트
- 항목별 강·약 프로파일: 레이더 차트 또는 항목별 막대. 강점은 강조색, 약점은 '개선 필요' 표식
- 약점 항목 카드마다 "이 활동 추천받기" 진입점(→ 활동 추천으로 연결)
- "무료로 정식 측정 받기" CTA(→ 센터 연계). 자가측정 결과였다면 정식 측정 유도 강조
- 등급별 평가 항목이 다름을 자연스럽게 안내(예: 새싹은 건강체력+BMI 기준). 미입력으로 판정 불가한 상위 등급은 "정식 측정 시 확인 가능"으로 부드럽게 처리

### 4.5 센터 연계 (CTR-01)
- 지역 선택 또는 현재 위치 기반, 가까운 체력인증센터 목록(거리순 카드): 센터명, 주소, 거리
- 각 센터 카드에 "유아 무료 측정 가능" 배지 + "예약하기"(국민체력100 예약 페이지 링크)
- 가능하면 지도 뷰 토글

### 4.6 기록 & 성장 추적 (REC-01, REC-02)
- 측정 기록 리스트: 날짜별, 정식/자가 구분 배지
- 성장 추이 그래프: 시계열. **정식값=실선·진한 점, 자가값=점선·흐린 점**으로 명확히 구분. 항목 선택 필터
- 점이 적을 때도 빈약해 보이지 않도록 축·기준선·등급 배경 밴드로 맥락 제공

### 4.7 활동 추천 (ACT-01)
- 약점 요소별로 그룹핑된 운동 콘텐츠 카드(썸네일 + 제목 + 대상 체력요소 태그)
- 진단 결과에서 넘어온 경우 해당 약점 요소가 상단에 우선 노출

### 4.8 지역 참여 인사이트 (INS-01, INS-02)
- 통찰 카드(핵심): "OO구의 유아 체력측정 참여는 전국 평균보다 낮은 편이에요. 가까운 센터에서 무료로 측정해보세요." — 한 문장 + 센터 연계 버튼
- 확장(P1, 여력 시): 시군구 참여율 지도(색상) 또는 랭킹 막대. 데이터 부족 시 카드만 노출로 폴백

## 5. 컴포넌트 시스템(공통)
- 등급 배지 컴포넌트(씨앗/새싹/꽃/열매, size 대/중/소)
- 측정 항목 입력 카드, 정식/자가 구분 배지, 강·약 태그, 추천 콘텐츠 카드, 센터 카드, 통찰 카드
- 빈 상태(Empty state): 측정 기록 없음, 지역 데이터 없음 등은 일러스트 + 다음 행동 안내로 친절하게

## 6. 접근성 & 카피 톤
- 카피는 학부모에게 말 걸듯 다정하고 쉬운 존댓말. 전문 용어(백분위 등) 대신 "또래 기준", "우리 아이 강점"처럼 풀어쓰기
- 진단 결과는 아이를 평가·비난하는 뉘앙스 배제. 성장·격려 프레임 유지("씨앗부터 열매까지 함께 키워가요")
- 색상만으로 정보 구분하지 않기(등급은 색+아이콘+텍스트 병기)
